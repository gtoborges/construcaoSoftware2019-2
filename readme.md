## Comandos para rodar o client

  - cd client/csgo-view
  - npm install
  - npm run serve

## Comandos para rodar o server

  - cd server
  - npm install
  - export NODE_ENV=development && npm run debug

## Comando para utilizar o dumb do banco de dados

  - mysql -u username -p csgo_app < db_dump.sql
